# Caracol

Nuxt 4 app scaffold using TypeScript, Tailwind CSS, and shadcn-vue.

## Quick Start

```bash
npm install
npm run dev
```

Runs on [http://localhost:7777](http://localhost:7777).

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run generate`
- `npm run lint`
